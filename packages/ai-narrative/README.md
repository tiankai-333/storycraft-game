# AI Narrative Engine

This package will become the reusable AI participation layer.

Responsibilities:

- generate narration from structured events
- generate constrained NPC dialogue
- summarize memory
- review narrative consistency
- present clues and foreshadowing

Non-responsibilities:

- deciding whether an action succeeds
- creating rooms or exits
- mutating inventory
- completing quests
- changing NPC life/death state

The game runtime decides facts. This package expresses them.

# Demo Adventure

First playable content pack.

Suggested premise:

```text
Seven-turn mystery: the master died on a snowy night.
```

Required content:

- 5-8 rooms
- 3 NPCs
- 5 items
- 1 main quest
- 3-5 clues
- 2-3 endings

Design goal:

Every meaningful action should reveal information, change trust, consume time, increase danger, or alter ending conditions.
